Info for Parents

Information for parents who are new to Troop 463

Scoutbook

The troop uses Scoutbook to manage our calendar and track advancement. It’s critical that all scouts are able to login to scoutbook and are doing so to update their requirements. This is achieved by using the same login as for https://my.scouting.org.

Follow this URL for Scoutbook: https://scoutbook.scouting.org/

This tutorial will help parents understand Scoutbook if they run into any issues with the website. https://help.scoutbook.scouting.org/knowledge-base/scoutbook-tutorial-for-parents/

Calendar

Everyone who wishes to subscribe to the troop calendar can import using the link below. For instructions, please see the Scoutbook tutorial above. https://api.scouting.org/advancements/events/calendar/52288

It is also possible to view the calendar at https://advancements.scouting.org. Login using my.scouting.org account and navigate to “calendar”.

The general calendar for outings will be shared in Parent Meeting notes.

Communication platforms

Discord

All scouts are encouraged to get Discord access for communication. Sign up here https://discord.com/ . For scouts under 13, they need to use an account managed by their parent. To do this, use an email account managed by a parent and sign up for your scout. Contact Debbie to be invited to the troop.

groups.io

There is a groups.io group that the troop uses primarily as a mailing list for parents. https://groups.io/g/BSATroop463

WhatsApp

There is a Troop 463 Parents WhatsApp group primarily for coordinating pickup/dropoffs on outing days. The group admins are Debbie, Kasuen, Lalit, Sarit.

SmugMug

The troop stores its event photos on SmugMug. Contact Monty for access.
